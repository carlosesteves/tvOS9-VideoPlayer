# TVPT

## What is it?

TVPT is an app for tvOS (Apple TV 4) which is able to render multiple portuguese channel streams: 
- RTP1
- RTP2
- RTP3
- SIC
- SIC NOTICIAS
- TVI
- TVI 24

## How to install?

1. Checkout the code
2. Build the APP using Xcode
3. Sideload to Apple TV 4
4. Launch the APP
